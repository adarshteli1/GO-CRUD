# GO CRUD

A CRUD (Create, Read, Update, Delete) application built with Go.

## Getting Started

Clone the repository:
```bash
git clone https://github.com/adarshteli1/GO-CRUD.git
cd GO-CRUD
```

## Features

- Create, Read, Update, and Delete operations
- Built with Go
- RESTful API design

## Installation

```bash
go get ./...
go run main.go
```
